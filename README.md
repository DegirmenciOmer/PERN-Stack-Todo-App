# PERN-Stack-Todo-App


## General Info
A basic todo app which has create, update, and delete features.
URL: https://pern-todo-omer.netlify.app/

## Technologies 
- Backend: NodeJS, Express
- Database: PostgreSQL
- Frontend: ReactJS
- UI: React Bootstrap
